# flow-chart
  简单的流程图绘制，前端页面，应用jquery.js、d3.js、semantic.css，功能包括：
  - 流程图块生成、拖拽、连线；
  - 放大缩小功能；
  - 导入导出json数据；
  - 产生相应的xml和xpdl；
  - 保存（还未完成，待更新）；
